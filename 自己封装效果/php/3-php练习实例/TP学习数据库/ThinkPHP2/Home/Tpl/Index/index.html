标量输出

hello my name is {$name1}

hello my name is {$name2}

数组输出

hello my name is {$name3}

hello my name is {$name3a}

hello my name is {$name3b[1]}

hello my name is {$name4}

hello my name is {$name4a}

hello my name is {$name4b[c3]}

hello my name is {$name4b.c3}

对象输出

hello my name is {$NF:f}

hello my name is {$NF->f}

系统变量

首先要有get的值

hello my name is {$name5}

hello my name is {$Think.get.name}

hello 浏览器的版本是 {$Think.version}

hello 获取常量是 {$Think.const.APP_NAME}

使用函数

hello 这里是经过md5加密的数据 {$name6|md5}

hello 这里是变为大写 {$name6|strtoupper}

hello 这里是时间戳 {$name7}

hello 这里是设置过格式的时间 {$name7|date="Y m d H:i:s",###}

hello 这里是设置过格式的时间 {$name7|date="Y-m-d",###}

设置默认值

hello 这里是设置默认值的方法 {$name8|default="我是默认值"}

hello 这里是设置默认值的方法 {$name8a|default="我是默认值"}

运算符使用

hello 这里是运算符的使用 {$num+1}

hello 这里是运算符的使用 {$num-1}

hello 这里是运算符的使用 {$num*3}
